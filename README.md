# 云网宽带微信小程序

## 项目简介

这是一个基于微信小程序开发的移动宽带安装服务平台,从Vue H5应用完整转换而来,保留了所有原有的样式和功能特性。

## 项目特性

- 📱 原生小程序体验,性能优越
- 🎨 完美保留H5版本的渐变背景和动画效果
- 🔐 完整的用户认证和角色切换流程
- 🌊 动态波浪背景和毛玻璃效果
- 📋 支持扫码绑定、业务办理等功能
- 🧪 内置测试工具和模拟数据

## 技术栈

- **框架**: 微信小程序原生开发
- **UI组件**: Vant Weapp 小程序组件库
- **样式**: WXSS + CSS3动画
- **图标**: SVG图标 + Emoji表情
- **设计稿**: 750rpx标准小程序设计稿

## 样式特性

### 保留的H5样式
1. **渐变背景**: 
   - 启动页: `#667eea → #764ba2 → #f093fb`
   - 登录页: `#2563eb → #3b82f6 → #8b5cf6`
   - 绑定页: `#2563eb → #16a0f6`

2. **动画效果**:
   - Logo入场动画 + 脉冲效果
   - 加载点弹跳动画
   - 波浪背景动画
   - 文字滑入动画

3. **毛玻璃效果**: 使用半透明背景+阴影模拟

4. **响应式设计**: 适配不同屏幕尺寸

### 单位转换
- H5: `px → vw` (375px设计稿)
- 小程序: `px → rpx` (750rpx设计稿)
- 转换比例: `1px ≈ 2rpx`

## 图标方案

### TabBar图标
- 使用SVG格式,支持高分辨率
- 包含选中/未选中两种状态
- 颜色: 未选中`#999`, 选中`#1d63ff`

### 功能图标
- 临时使用Emoji表情(🛒📄💬等)
- 可后续替换为iconfont图标库
- 保持与H5版本功能一致性

## 开发说明

### 环境要求
- 微信开发者工具
- Node.js (可选,用于构建工具)

### 快速开始
1. 用微信开发者工具打开`cloud_app`目录
2. 配置AppID(可使用测试号)
3. 点击编译运行

### 配置说明
- `project.config.json`: 项目配置,包含AppID
- `app.json`: 全局配置,包含页面路由和TabBar
- `sitemap.json`: 搜索配置
