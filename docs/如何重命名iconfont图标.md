## 当前状态

✅ 已更新链接：`//at.alicdn.com/t/c/font_5075390_r931kta192.css`

⚠️ 图标类名仍然是自动生成的（如 `icon-a-320655539`），需要重命名为有意义的名称。

## 重命名步骤

### 步骤 1：登录 iconfont
访问 https://www.iconfont.cn/ 并登录你的账号

### 步骤 2：进入项目
1. 点击顶部导航栏的"资源管理"
2. 选择"我的项目"
3. 找到项目 ID 为 **5075390** 的项目
4. 点击进入项目

### 步骤 3：重命名图标

对于项目中的每个图标：

1. **找到图标**：在项目页面中找到要重命名的图标
2. **点击编辑**：将鼠标悬停在图标上，点击右上角的"编辑"按钮（铅笔图标）
3. **修改 Font Class**：
   - 在弹出的编辑框中找到"Font Class"输入框
   - 输入新的类名（**不要加 `icon-` 前缀**）
   - 例如：输入 `gouwuche`，系统会自动生成 `icon-gouwuche`
4. **保存**：点击"保存"按钮

### 步骤 5：更新代码

重命名完成后：

1. **点击"更新代码"**：在项目页面右上角找到"更新代码"按钮并点击
2. **查看在线链接**：点击"Font class" → "查看在线链接"
3. **复制新链接**：复制生成的新 CSS 链接
4. **告诉我新链接**：将新链接发给我，我会更新项目代码

## 注意事项

### ⚠️ 重要提示

1. **Font Class 不要加前缀**
   - ❌ 错误：输入 `icon-gouwuche`
   - ✅ 正确：输入 `gouwuche`
   - 系统会自动添加 `icon-` 前缀

2. **使用拼音命名**
   - 便于记忆和使用
   - 避免使用特殊字符
   - 建议全部小写

3. **每次修改后要更新代码**
   - 修改 Font Class 后，必须点击"更新代码"
   - 否则生成的 CSS 文件不会更新
   - 链接的时间戳会改变

4. **检查图标数量**
   - 首页需要 15 个图标
   - 当前只有 8 个图标
   - 还需要添加 7 个图标

## 常见问题

### Q1: 为什么我修改了名称，但生成的 CSS 还是旧的？
**A**: 修改后必须点击"更新代码"按钮，否则不会生成新的 CSS 文件。

### Q2: 我可以批量重命名吗？
**A**: iconfont 不支持批量重命名，需要逐个编辑。

### Q3: 重命名后需要重新下载字体文件吗？
**A**: 不需要，我们使用的是在线 CDN，只需要更新链接即可。

### Q4: 类名可以使用中文吗？
**A**: 不建议，建议使用拼音或英文，避免编码问题。
